### 五、项目介绍
项目编号：python109D

基于Python+Django的网络作品管理系统，方便用户随时随地浏览和管理文学作品，提升系统运营质量。

角色：管理员、用户

用户功能：注册登录、作品浏览、作品搜索、作品详情、章节阅读、作品收藏、作品评论、个人中心。

管理员功能：登录、用户管理、作品分类管理、作品信息管理、章节管理、评论管理、阅读榜单管理、系统公告管理。

数据库表数量：12张表。
